# 📖 About CoProT

CoProT is **not another coding platform**.

Instead of replacing platforms like **LeetCode**, **GeeksforGeeks**, **Codeforces**, or **HackerRank**, CoProT acts as a **central productivity and learning companion** for programmers preparing for coding interviews.

It helps developers organize their entire coding journey in one place by tracking solved problems, storing personal learning notes, analyzing strengths and weaknesses, and providing meaningful insights that existing coding platforms often don't focus on.

---

# 🎯 The Problem

During interview preparation, developers usually solve problems on multiple platforms.

- LeetCode
- GeeksforGeeks
- Codeforces
- HackerRank
- InterviewBit

As a result,

- progress becomes scattered,
- mistakes are forgotten,
- revision becomes difficult,
- learning patterns are never analyzed.

CoProT solves this problem.

---

# 💡 Vision

> **Build a personal coding productivity platform that helps developers capture, organize, and analyze their coding journey across multiple platforms.**

# 🚀 Getting Started

Clone the repository

```bash
git clone <repo-url>
```

Move inside

```bash
cd CoProT
```

Create virtual environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python backend/app.py
```
